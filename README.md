# semanaOnministack11
Be the Hero
- App feito com base no curso da Rockeseat, onde era para ser criado um sistema onde ONGs poderiam acessar, criar uma conta
  Inserir causas onde pessoas poderiam ajudar entrando em contato via whatsapp ou email.

- Foi criado o backend em NodeJS usando express para fazer a API e o KNEX para gerenciar o banco de dados(postgresql)

- Na Web foi usado React, e o axios para consumir a API criada, na parte web é possível logar com a ong e ver os casos criados e criar novos casos
alem de criar uma conta.

- No mobile foi usado React Native, e suas funções eram basicamente de visualizar os casos disponíveis e clicar para ajudar, porém coloquei as funções já existentes na parte web no mobile, assim o usuário pode fazer login e ter acesso a tudo em um lugar só.

<h3>Esse projeto serviu para praticar e conhecer mais do react native que 
é uma stack muito interessante e poderosa</h3>
